# Microsoft 365 SaaS Administration Lab

## Overview
Hands-on Microsoft 365 administration lab demonstrating real enterprise
SaaS management skills used daily by IT helpdesk and system administrators.

## Environment
- Platform: Microsoft 365 E5 Developer Tenant
- Users: 25 licensed accounts
- Services: Exchange Online, Teams, SharePoint, Azure AD

## Skills Demonstrated
- Azure Active Directory user provisioning and management
- Microsoft 365 license assignment and management
- Security group and distribution list administration
- Multi-Factor Authentication configuration
- Conditional Access policy creation
- Exchange Online mailbox management
- SharePoint site administration
- Microsoft 365 security and compliance center

## Real World Relevance
These tasks represent daily responsibilities at organizations running
Microsoft 365 which includes over 300 million commercial users worldwide.
Every IT helpdesk technician working in a modern enterprise environment
manages Microsoft 365 on a daily basis.
