# Phishing Simulation Lab

## Overview
Authorized phishing simulation lab using GoPhish framework to practice
social engineering assessment techniques used in real penetration testing
engagements. All campaigns conducted against self-owned test accounts in
a controlled environment.

## Purpose
Social engineering accounts for over 90% of successful breaches. This lab
practices the methodology pentesters use to assess human vulnerability
in organizations including email template creation, campaign tracking,
and professional reporting.

## Tools Used
- GoPhish — Open source phishing simulation framework
- Custom HTML email templates
- Landing page cloning

## Legal Notice
All phishing simulations conducted exclusively against self-owned accounts
in a controlled lab environment. No real users targeted.
