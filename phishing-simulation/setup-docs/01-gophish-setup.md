# GoPhish Lab Setup

## Date
May 21, 2026

## What Is GoPhish
GoPhish is an open source phishing simulation framework used by penetration
testers to assess organizational susceptibility to phishing attacks. It provides
campaign management, email tracking, credential harvesting, and detailed
reporting.

## Legal Scope
All campaigns in this lab target self-owned Gmail test accounts only.
No real users or organizations targeted.

## Installation
- Platform: Windows 11
- GoPhish version: Latest stable release
- Admin interface: https://localhost:3333

## Lab Objectives
- Build realistic phishing email templates
- Create convincing landing pages
- Run simulated campaigns against test accounts
- Analyze click rates and credential submission
- Generate professional phishing assessment reports
