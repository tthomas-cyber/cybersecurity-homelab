# Cybersecurity Home Lab
**Author:** Tyler Thomas  
**GitHub:** tthomas-cyber  
**Focus:** Penetration Testing | Network Security | Vulnerability Assessment

## Overview
A personal cybersecurity research lab built to develop hands-on penetration testing skills. This repository documents tools, methodology, and findings from controlled lab environments using TryHackMe, VulnHub, and local vulnerable VMs.

## Environment
- **Attack Platform:** Kali Linux (WSL2)
- **Practice Targets:** TryHackMe, Metasploitable 2
- **Tools:** Nmap, Metasploit, Gobuster, Nikto, Wireshark, Netcat
- **Documentation:** Markdown

## Repository Structure
- **/tools-and-methodology** — Tool usage notes and attack methodology
- **/assessments** — Vulnerability assessment reports
- **/writeups** — TryHackMe and HackTheBox room writeups
- **/notes** — General study notes and references
