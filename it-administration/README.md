# IT Administration Lab

## Overview
Hands-on Active Directory administration lab simulating real enterprise
IT tasks including user lifecycle management, organizational unit design,
security group management, and Group Policy configuration.

## Environment
- **Domain:** corp.local
- **Domain Controller:** Windows Server 2019 (DC01)
- **Tools:** Active Directory Users and Computers, Group Policy Management Console, PowerShell

## Skills Demonstrated
- Active Directory user and group management
- Organizational Unit design and delegation
- Group Policy Object creation and linking
- Security group permission management
- PowerShell automation of AD tasks
- User account lifecycle management

## Real World Relevance
These tasks represent daily responsibilities of IT support specialists,
system administrators, and helpdesk technicians in enterprise environments.
Every organization running Windows uses Active Directory for identity
and access management.
